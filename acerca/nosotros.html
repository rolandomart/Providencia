GIRASOLES “LA PROVIDENCIA.”

1. OBJETIVOS DEL PROYECTO Los objetivos claves del proyecto son: • Mejorar las ventas y aumentar el número de producción en la providencia y así poder implementar un sitio web para la propagación de publicidad y la gente tenga el conocimiento del lugar. • para facilitar que el mismo establecimiento se den las relaciones de compra y venta.

2. CARÁCTER DEL PROYECTO El proyecto tiene dos componentes mayores: • Identidad Corporativa. • Y Sitio Web. Estos dos se dividen en más ramas y cada conlleva un proceso diferente para la realización y terminación, por lo cual los tiempos estan definidos junto a la parte de las fases del proyecto. Este proyecto fue implementado para la venta y promoción de girasoles la providencia. 3. DESCRIPCIÓN DEL PROYECTO Cada participante será pedido hacer un compromiso que abarca a asistencia. Además, su actividad provisional será facilitada con discusiones en Internet. Y cada fase dura un cuatrimestre.
